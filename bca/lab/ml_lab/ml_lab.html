Machine Learning Lab

Programs List

1. FIND-S Algorithm
Implement and demonstrate the FIND-S algorithm to find the most specific hypothesis given training samples. Read training data from a .csv file and display the evolving hypothesis.
2. Candidate-Elimination Algorithm
For a set of training examples in a .csv, implement Candidate-Elimination to compute the version space (S and G boundaries) and output all hypotheses consistent with the examples.
3. Decision Tree (ID3)
Write a program to build a decision tree using the ID3 algorithm on an appropriate dataset (CSV), then use the tree to classify new samples and report the classification result.
4. Naive Bayes Classifier (numeric/categorical)
Implement a Naive Bayesian classifier for a training dataset in CSV. Train the model, classify test sets, and compute accuracy (and optionally confusion matrix).
5. Naive Bayes for Document Classification
Use Naive Bayes to classify a collection of documents (preprocess text, TF or TF-IDF). Compute accuracy, precision and recall on test data.
6. Bayesian Network for Medical Diagnosis
Construct a Bayesian network (e.g., for heart disease) using a medical dataset (Heart Disease dataset). Use it to perform diagnosis/inference and evaluate results.
7. Expectation-Maximization (EM) Clustering
Apply the EM algorithm to cluster data from a CSV file (e.g., Gaussian Mixture Model). Show cluster assignments and basic evaluation (log-likelihood, cluster sizes).
8. k-Means Clustering
Use k-Means to cluster a dataset from CSV. Visualize clusters (if 2D/3D), show inertia/within-cluster sum-of-squares and compare results for different k values.
9. k-Nearest Neighbour (k-NN) on IRIS
Implement k-NN to classify the IRIS dataset. Print both correct and wrong predictions and compute accuracy, precision, recall per class.
10. Locally Weighted Regression (Non-parametric)
Implement Locally Weighted Regression to fit data points (select appropriate dataset). Plot fitted curve vs data and report error metrics.

Reverse a Decimal Number

Algorithm

  1. Start
  2. Read an integer n
  3. Initialize reverse = 0
  4. Repeat while n ≠ 0:
    • Extract remainder: rem = n % 10
    • Update reverse: reverse = reverse × 10 + rem
    • Reduce number: n = n / 10
  5. Print reversed number
  6. Stop

Program

#include<stdio.h>
void main()
{
    int n, reverse=0, rem;
    printf("Enter a number:\n");
    scanf("%d", &n);

    while(n!=0)
    {
        rem=n%10;
        reverse=reverse*10+rem;
        n/=10;
    }

    printf("Reversed Number: %d\n",reverse);
}

Output

Enter a number:
12345
Reversed Number: 54321

Compute Nth Fibonacci Number

Flowchart

(Replace when image available)

Algorithm

  1. Start
  2. Read value of n
  3. Initialize previousFib = 0 and currentFib = 1
  4. Repeat loop from i = 0 to n - 2
  5. Compute newFib = previousFib + currentFib
  6. Update previousFib = currentFib and currentFib = newFib
  7. Display currentFib
  8. Stop

Program

#include<stdio.h>

void main()
{
    int i,n,newFib,previousFib = 0, currentFib = 1;
    printf("Enter n\n");
    scanf("%d",&n);

    for(i=0;i<n-1;i++)
    {
        newFib = previousFib + currentFib;
        previousFib = currentFib;
        currentFib = newFib;
    }

    printf("Computing Fibonacci number is = %d",currentFib);
}

Output

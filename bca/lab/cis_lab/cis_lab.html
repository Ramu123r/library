Cryptography & Information Security Lab

Programs List

1. XOR each character (C)
Write a C program that contains a string (char pointer) with value "Hello World". The program should XOR each character in this string with 0 and display the result.
2. AND/XOR with 127 (C)
Write a C program that contains a string (char pointer) with value "Hello World". The program should AND or XOR each character in this string with 127 and display the result.
3. Classical Ciphers (Java)
Write a Java program to perform encryption and decryption using:\n• Caesar Cipher\n• Substitution Cipher\n• Hill Cipher
4. DES algorithm (Java)
Write a Java program to implement the DES algorithm logic (educational implementation / demonstration of rounds, S-boxes, permutations).
5. Blowfish (C/Java)
Write a C or Java program to implement the Blowfish algorithm logic (key schedule and round function demonstration).
6. Rijndael / AES (C/Java)
Write a C or Java program to implement the Rijndael (AES) algorithm logic (SubBytes, ShiftRows, MixColumns, AddRoundKey).
7. RC4 & Blowfish (Java)
Write RC4 logic in Java. Using Java Cryptography, encrypt text "Hello world" using Blowfish. Create your own key using Java keytool (instructions/demonstration).
8. RSA Algorithm (Java)
Write a Java program to implement the RSA algorithm (key generation, encryption, decryption) and demonstrate with sample plaintext.
9. Diffie–Hellman (HTML + JS)
Implement the Diffie–Hellman Key Exchange mechanism using HTML and JavaScript (demonstrate generation of shared secret between two parties).
10. SHA-1 Message Digest (Java)
Calculate the message digest of a text using the SHA-1 algorithm in Java (show hex digest output and sample verification).
11. MD5 Message Digest (Java)
Calculate the message digest of a text using the MD5 algorithm in Java (show hex digest output and sample verification).
